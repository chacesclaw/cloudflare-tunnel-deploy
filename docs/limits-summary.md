# Cloudflare 免费版限制说明

## 📊 流量与带宽

| 项目 | 免费版 | 说明 |
|------|--------|------|
| 总月度流量 | **无硬性上限**（Fair Use 原则） | 个人站不会触发限制 |
| 单文件上传 | **最大 100MB** | 超过返回 413 错误 |
| 带宽速率 | 不限速 | 源站多快 CDN 多快 |
| Tunnel 数量 | 无限 | 免费版不限制 |

## ⚠️ 需要留意的限制

- **100MB 文件上传限制**：如果需要上传大文件，可能无法通过 Tunnel
- **Fair Use 政策**：持续大量消耗带宽（如视频流），Cloudflare 可能要求升级
- **Page Rules 最多 3 条**：如果有多条重定向规则需要注意数量
- **缓存 500MB/区域**：免费版 CDN 节点缓存空间有限

## 总结

个人博客、展示站、小应用：**免费版完全够用**。
