## 📌 核心知识点：六位数字 .xyz 为什么便宜？

XYZ 注册局（XYZ.COM LLC）针对 **6 位及以上纯数字 .xyz 域名** 推出了长期低价政策：

> **六位纯数字 .xyz = 永久低价，续费与首年同价**

这是因为：
- 纯数字域名无法用于品牌建站（看起来像 spam）
- 注册局干脆低价卖给搞技术的人用
- 所以这**不是首年促销坑**，而是注册局的独立定价体系

> **非数字 .xyz 不享受此优惠**，续费回到 $10~15/年。

## 🛒 方案 A：阿里云购买（国内用户推荐，支持支付宝）

### 第一步：打开阿里云域名注册

打开 https://www.aliyun.com/ → 搜索"域名注册" → 进入万网

或者直接访问：https://mi.aliyun.com/

### 第二步：搜索六位数字

在搜索框输入 **6 位以上的纯数字**，后缀选 **.xyz**：

```
例如：5201314.xyz、314159.xyz、888888.xyz
```

> 💡 **随便编个有意义的数字组合**：
> - 生日：`199001.xyz`、`1205.xyz`
> - 幸运数字：`888888.xyz`、`666666.xyz`
> - 纪念日：`0520.xyz`
> - 随机的：`9527.xyz`、`10086.xyz`

### 第三步：确认价格

搜索结果中会显示：
- 域名状态：**"可注册"**（绿色勾）
- 价格：**7 元/年**（六位数字 .xyz 专属价）
- 续费价格：**7 元/年**

> ⚠️ **如果显示价格不是 7 元/年**，说明这个域名不是六位纯数字（比如包含了字母），续费会很贵。换一个数字组合试试。

### 第四步：加入购物车并购买

1. 点击 **"加入清单"**
2. 在购物车中，**年限选择"10 年"**
3. 总价显示：**7 元 × 10 年 = 70 元**
4. 点击 **"立即购买"**
5. 如果没有阿里云账号，先注册（需要手机号验证）
6. 域名持有者信息填写（个人身份信息）
7. 勾选 **"开启自动续费"**（锁住 7 元/年的续费价）
8. 选择 **支付宝** 支付 → 扫码付款

### 第五步：实名认证（中国要求）

根据工信部要求，国内注册的域名需要实名认证：

1. 在阿里云控制台 → **域名列表**
2. 找到刚买的域名 → **"实名认证"**
3. 上传身份证照片（个人）或营业执照（企业）
4. 审核通常 1~3 个工作日

> **实名认证通过前域名不能使用**，建议先提交认证，等通过了再继续配置。

## ❌ 常见问题

### Q：六位数字 .xyz 看起来像垃圾域名，能用吗？

在 Cloudflare 和本项目的场景下，域名只是**用来路由流量**的。数字 .xyz 和字母 .com 在技术上**没有任何区别**——Cloudflare 不会因为它是一串数字就区别对待。

> 就像 `127.0.0.1` 也是数字，但它是最重要的 IP 地址。域名也是一样。

### Q：续费真的和首年一样便宜吗？

**六位纯数字 .xyz 是的。** 这是注册局的统一定价政策，Spaceship 和阿里云均承诺续费同价。不过为了保险，建议一次性买 5~10 年，锁定十年不变的价格。

### Q：在平台 A 买了，以后能转到 Cloudflare 管理吗？

**可以！** 域名注册商（买域名的地方）和 DNS 托管商（管理解析的地方）是两回事。你在阿里云/Spaceship 买了域名后，去 Cloudflare 添加域名，把 NS（域名服务器）改成 Cloudflare 的，以后 DNS 都在 Cloudflare 管理。

### Q：.xyz 域名在国内能用吗？

可以是。但如果用国内注册商（如阿里云），需要进行**实名认证**。如果走国外注册商（Spaceship），则不需要。

本项目使用 Cloudflare Tunnel，域名只需要在 Cloudflare 上托管，和国内备案没有关系。

### Q：六位数字 .xyz 会不会被注册局回收？

不会。一旦你购买了域名，在续费期内它就是你的合法资产，注册局不会因为它是"特价数字"就单方面收回。这和普通域名的所有权是一样的。

### Q：怎么判断我搜的域名是不是六位数字？

看搜索结果中的**价格**：
- **7 元/年（阿里云）或 $0.60/年（Spaceship）** → ✅ 六位数字 .xyz
- **$9+/年（≈ 65 元+）** → ❌ 这是普通 .xyz，续费贵
- 域名字符串中**只要包含字母** → ❌ 不是纯数字
