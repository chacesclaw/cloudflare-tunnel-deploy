## ❌ 常见问题

### Q：.xyz 域名是不是不太正经？

.xyz 是一个标准的顶级域名（TLD），和 .com、.net 是同一级别的。早期因为很多垃圾邮件注册 .xyz 导致口碑差，但后来 Google 收购了 abc.xyz 等知名品牌使用后，.xyz 的接受度已经大幅提高。

在 Cloudflare 和本项目面前，.xyz 和 .com **没有任何功能上的区别**。

### Q：在 Namesilo 买域名，后面能转到 Cloudflare 管理吗？

**可以！** 域名注册商（Namesilo）和 DNS 托管商（Cloudflare）是两回事。你在 Namesilo 买了域名后：

1. 在 Cloudflare 添加域名
2. 把 Namesilo 的 NS（Name Server，域名服务器）改成 Cloudflare 的
3. 以后 DNS 都在 Cloudflare 管理
4. 域名本身（续费、转移等）还在 Namesilo

**详细操作见**：[域名托管到 Cloudflare 教程](./domain-setup-cloudflare.md)

### Q：.xyz 域名在国内能用吗？

.xyz 是经过工信部备案审核的顶级域名，可以在国内使用。但如果需要**备案**（针对国内服务器），各地区的备案政策可能有差异，建议咨询你的服务器提供商。

> 不过本项目用 Cloudflare Tunnel，它要求域名在 Cloudflare 上托管，和国内备案没有关系。

### Q：优惠码在哪里输入？

在购物车页面的 **"Have a Coupon or Promotion Code?"** 文本框输入，点击 **Apply**。如果输入正确，价格会变化。
